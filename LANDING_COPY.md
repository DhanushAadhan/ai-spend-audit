## FAQ Section

**Q: Is my data safe?**
A: Your spend data is used only to calculate savings. We never sell it or share it with AI vendors.
Shareable URLs strip all personal information.

**Q: How accurate are the savings estimates?**
A: All pricing comes directly from vendor websites, verified weekly. Enterprise pricing is estimated
from public sources and clearly marked. See PRICING_DATA.md for all sources.

**Q: Do I need to connect any accounts?**
A: No. You enter your spend data manually. No OAuth, no API keys, no bank connections.

**Q: What AI tools do you cover?**
A: Cursor, GitHub Copilot, Claude, ChatGPT, Anthropic API, OpenAI API, Gemini, and Windsurf.
More tools added based on user requests.

**Q: Is AuditAI really free?**
A: Yes. The core audit is free forever. We may add Pro features for teams in the future.
